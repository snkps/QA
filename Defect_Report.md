# Звіт про документування дефектів (Вебзастосунок: Book.ua)

Цей звіт містить опис функціональних та візуальних дефектів, знайдених під час тестування сайту Book.ua, згідно з лабораторною роботою №6.

| Headline | Severity | Опис | Фактичний результат | Очікуваний результат | Примітка |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Частковий переклад інтерфейсу** | **Major** | Зайти на сайт, обрати англійську мову. | Частина елементів лишається українською. | Весь інтерфейс має перекладатись англійською. | Порушення локалізації, впливає на UX. (Див. Defect_Translation.png) |
| **Відсутня біографія автора** | **Average** | Пошук автора, відкрити профіль автора. | Відображається тільки ім'я. | Очікується коротка біографія автора. | Типова проблема у більшості профілів авторів. (Див. Defect_Author_Bio.png) |
| **Відсутня кнопка "Головна"** | **Minor** | Відкрити сайт, перейти в розділи, спробувати повернутись на головну. | Відсутня можливість швидкого повернення. | Має бути кнопка або іконка для повернення на головну сторінку. | UX-покращення для навігації. (Див. Defect_Author_Bio.png) |
| **Відсутні відгуки користувачів** | **Average** | Відкрити сторінку будь-якої книги, перейти до розділу "Відгуки". | Відгуки відсутні, лише форма для написання. | Очікується відображення відгуків інших користувачів. | Зниження інформативності для користувача. (Див. Defect_No_Reviews.png) |
